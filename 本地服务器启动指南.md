## 🔧 故障排除

### 常见问题

#### 问题1：端口8000被占用
**现象**：启动时提示端口8000已被使用
**解决**：
```bash
# 查看占用端口的进程
netstat -ano | findstr :8000

# 杀死进程（替换PID为实际进程ID）
taskkill /PID [进程ID] /F
```

#### 问题2：Python未找到
**现象**：提示Python不是内部或外部命令
**解决**：
1. 安装Python 3.7+
2. 确保Python添加到PATH环境变量
3. 重启命令行

#### 问题3：桌面版无法启动
**现象**：双击启动桌面游戏.bat无反应
**解决**：
1. 检查Python环境是否正确安装
2. 查看是否有pygame等依赖包
3. 在命令行中手动运行查看错误信息

#### 问题4：Web版无法访问
**现象**：浏览器显示无法连接
**解决**：
1. 确认后端服务器已启动
2. 检查防火墙设置
3. 确认端口8000未被其他程序占用
