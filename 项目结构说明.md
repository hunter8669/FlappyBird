# FlapPy Bird 项目结构说明

## 📁 核心目录结构

```
FlapPyBird-master1/
├── 📁 backend/                    # 后端服务器
│   ├── simple_server.py          # 主服务器（含管理后台）
│   ├── download_stats.json       # 下载统计数据
│   ├── requirements.txt          # Python依赖
│   └── app/                       # FastAPI应用（备用）
│
├── 📁 frontend/                   # 前端网站
│   ├── index.html                # 主页面
│   ├── src/                      # React源码
│   ├── public/                   # 静态资源
│   └── package.json              # Node.js依赖
│
├── 📁 game-desktop/              # 桌面版游戏
│   ├── main.py                   # 游戏入口
│   ├── game/                     # 游戏核心代码
│   ├── assets/                   # 游戏资源
│   └── README.md                 # 游戏说明
│
├── 📁 src/                       # 备用游戏代码
│   └── flappy.py                 # 单文件游戏版本
│
├── 📁 services/                  # 微服务架构（扩展用）
│   ├── user-service/            # 用户服务
│   ├── game-service/            # 游戏服务
│   └── analytics-service/       # 分析服务
│
├── 📁 docs/                      # 项目文档
├── 📁 output/                    # 项目输出文件
└── 📁 data/                      # 数据文件

## 🚀 启动脚本

├── 启动游戏网站.bat              # 启动Web服务器（推荐）
└── 启动桌面游戏.bat              # 启动桌面游戏

## 📄 项目文件

├── README.md                     # 项目说明
├── LICENSE                       # 开源协议
├── FlappyBird.zip               # 游戏压缩包
├── 软件管理大作业第三组--FlapPyBird小游戏.doc  # 项目文档
└── 项目历时表.xlsx               # 项目时间表

## 🔧 配置文件

├── .gitignore                   # Git忽略文件
├── pyproject.toml              # Python项目配置
├── Makefile                    # 构建脚本
└── .pre-commit-config.yaml     # 代码检查配置

## 🎯 快速使用

### 启动Web服务（推荐）
双击运行 `启动游戏网站.bat`
- 前端网站: http://localhost:8000
- 管理后台: http://localhost:8000/admin （admin/admin123）

### 启动桌面游戏
双击运行 `启动桌面游戏.bat`

## 📊 管理后台功能

- 🎮 游戏版本控制
